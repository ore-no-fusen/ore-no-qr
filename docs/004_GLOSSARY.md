# 004 用語集 (GLOSSARY)

本プロジェクト（俺のQR）で用いられる固有の用語、または特定の文脈で用いられる用語の定義。

### 俺のQR (Ore-no-QR)
本アプリケーションの名前。URLからシンプルな付箋デザインのQRコード画像を生成するWebアプリ。

### 俺の付箋 (Ore-no-Fusen)
本プロジェクトの姉妹アプリであり、インスピレーション元。PC向けの高速思考キャンバスアプリ。本アプリ（俺のQR）のプロモーションも兼ねて作成された。

### File System Access API
ブラウザ上で、ユーザーのローカルファイルシステムに直接アクセスしてファイルを読み書きするためのモダンWeb API。本アプリでは `showSaveFilePicker` メソッドを使用して、画像を保存する際の「名前を付けて保存」ダイアログを呼び出している。

### Firestore
Google Firebase が提供する NoSQL クラウドデータベース。本アプリでは、アクセス数と保存数の集計データを格納するために使用している。

### Vercel
フロントエンドアプリケーションのホスティングプラットフォーム。GitHubリポジトリと連携し、自動的にビルドとデプロイを行う。本アプリの本番環境（`ore-no-qr.vercel.app`）として使用されている。

### Vite
フロントエンド向けのビルドツール。本アプリおよびドキュメント(VitePress)のビルドエンジンとして使用されている。

### VitePress
Viteをベースにした静的サイトジェネレーター。本ドキュメントサイトを構築するために使用されている。Markdownファイルから高速なドキュメントページを生成する。
